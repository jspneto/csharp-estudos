# Estudos em C# - Programação Orientada a Objetos

Este repositório funciona como um **índice de estudos em C#**, reunindo pequenos projetos desenvolvidos ao longo do aprendizado de **programação orientada a objetos**.

Cada projeto é armazenado em um repositório independente, permitindo acompanhar sua evolução por meio de commits e tags do Git.

O foco não é apenas resolver os problemas, mas **registrar a transição do código procedural para soluções orientadas a objetos**, aplicando gradualmente conceitos como:

- Classes e objetos
- Atributos e métodos
- Encapsulamento
- Organização de código
- Boas práticas progressivas

## Projetos

### [Maior Área Entre Dois Triângulos][1]
- **Descrição:** Cálculo e comparação da área de dois triângulos usando a fórmula de Heron.
- **Conceitos abordados:** Procedural → Classes → Métodos.

### [Controle de Estoque Básico][2]
- **Descrição:** Exibe os dados de um produto e permite adicionar e remover os itens em estoque.
- **Conceitos abordados:** Classes/Métodos → Override.

## Organização e Versionamento

Cada projeto possui seu próprio versionamento baseado em tags:

- `v0.x` — Evolução conceitual inicial
- `v1.x` — Estrutura orientada a objetos mais consolidada

As tags registram **mudanças conceituais**, não apenas correções de código.

## Objetivo do repositório

Este índice serve como:
- Referência pessoal de estudo
- Histórico de aprendizado
- Material de consulta futura
- Registro público de evolução técnica

Nada aqui nasce perfeito — a ideia é **melhorar conscientemente a cada versão**.
